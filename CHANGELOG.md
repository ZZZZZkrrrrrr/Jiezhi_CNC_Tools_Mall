# 更新日志

本项目遵循语义化版本：

- `v1.x.x`：兼容更新、页面优化和新增功能。
- `v2.0.0`：可能包含不兼容的架构或业务调整。

## v1.0.1 — 2026-07-22

- 修复两份 Codex 执行方案文档在 GitHub 中显示乱码的问题。
- 文件统一重命名为 `Codex_数控精密刀具商城_网页端复刻执行方案.md/.docx`。

- 完成杰帜数控刀具商城高保真响应式网页。
